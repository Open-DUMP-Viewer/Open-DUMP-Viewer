SQL エクスポート

操作手順

  1. テーブル一覧でエクスポートしたいテーブルを選択します。
  2. ツールバーの SQL ボタンをクリック、 または オブジェクト(&O) > エクスポート > スクリプト(&S) を選択します。
  3. 保存先のファイルパスを指定します。
  4. エクスポートが実行されます。

対応 DBMS

INSERT 文は以下の DBMS 向けに生成できます:

DBMS構文の違い
OracleTO_DATE 関数, NUMBER 型そのまま
PostgreSQLCAST 構文, TIMESTAMP 型
MySQLバッククオート識別子, DATETIME 型
SQL Serverブラケット識別子, DATETIME2 型

オプション

  • DROP TABLE + CREATE TABLE 出力 (デフォルト: ON) — INSERT 文の前に DROP TABLE IF EXISTS と CREATE TABLE DDL を出力します。 対象データベースに同名テーブルが既に存在する場合、削除してから再作成されます。
  • 日付書式 — リテラル形式またはファンクション呼び出し

DROP TABLE の DBMS 別構文

DBMS構文
Oracle / PostgreSQLDROP TABLE IF EXISTS "テーブル名" CASCADE;
MySQLDROP TABLE IF EXISTS `テーブル名`;
SQL ServerIF OBJECT_ID('テーブル名', 'U') IS NOT NULL DROP TABLE [テーブル名];
注意: LOB データ (BLOB/CLOB) は SQL エクスポートに含まれません。 LOB データをエクスポートするには LOB ファイル抽出 を使用してください。
