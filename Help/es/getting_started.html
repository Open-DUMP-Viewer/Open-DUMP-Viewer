¿Qué es Open DUMP Viewer for Oracle database?

Open DUMP Viewer for Oracle database es una herramienta independiente que permite explorar y exportar archivos de volcado de bases de datos Oracle (EXP / EXPDP) sin necesidad de un cliente Oracle.

Características principales

  • Análisis de archivos de volcado — Compatible con los formatos EXP (heredado) y EXPDP (Data Pump)
  • Lista de tablas — Muestra esquemas, nombres de tablas e información de columnas en una vista de árbol
  • Vista previa de datos — Visualización paginada de datos de tablas con ordenación y copia
  • Búsqueda avanzada — Filtra datos con 12 operadores y condiciones compuestas AND/OR
  • Exportación CSV — Exporta datos de tablas a archivos CSV
  • Exportación SQL — Genera sentencias INSERT (Oracle / PostgreSQL / MySQL / SQL Server)
  • Exportación a Excel — Exporta a formato .xlsx / .xls
  • Exportación a Access — Exporta a bases de datos .mdb / .accdb
  • Exportación a SQL Server — Exporta directamente a una base de datos SQL Server
  • Exportación ODBC — Exporta a cualquier base de datos compatible con ODBC
  • Exportación por lotes — Exporta múltiples tablas a CSV/SQL de una sola vez
  • Extracción de archivos LOB — Extrae datos BLOB/CLOB/NCLOB a archivos individuales
  • Espacio de trabajo — Guarda y restaura el estado de trabajo
  • Exclusión de tablas — Oculta tablas no deseadas (con soporte de deshacer/rehacer)
  • Soporte multilingüe — 10 idiomas incluyendo japonés, inglés, chino, coreano, alemán, francés, español, italiano, ruso y portugués (BR)
  • Informes de errores — Reporta errores directamente a GitHub Issues desde la aplicación
  • Actualización automática — Detecta nuevas versiones y ofrece actualizaciones con un solo clic

Formatos de volcado compatibles

FormatoHerramientaEstado de compatibilidad
EXP (Heredado)exp / impCompatible (Oracle 7–11g)
EXPDP (Data Pump)expdp / impdpCompatible (Oracle 10g–23ai)
EXPDP Comprimidoexpdp COMPRESSION=ALLParcial (solo definiciones de tablas)

Instalación

  1. Ejecute el instalador MSI distribuido.
  2. Siga las instrucciones del asistente de instalación.
  3. Inicie "Open DUMP Viewer for Oracle database" desde el escritorio o el menú Inicio.
